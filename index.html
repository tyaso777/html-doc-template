Use This Template

Edit fragments under chapters-src/, keep shared shells under layouts/, and run python3 scripts/build_site.py to generate chapters/. Use examples/content-example.html as a reference for body content.

Replace the title, document information, body content, Materials, External Links, and Changelog. The Python runner loads Pyodide from a CDN only after the reader presses Load Python Runtime.

Included Capabilities

  • Auto-generated table of contents from data-toc elements
  • Prism code highlighting for Python, PowerShell, SQL, and Mermaid
  • Copy buttons for static code blocks and the Python runner editor
  • MathJax inline and display math
  • Mermaid diagram rendering
  • Optional Pyodide Python runner in a Web Worker
  • Document metadata and Changelog sections for maintenance
  • Split-template authoring workflow with shared assets
