First Section

Use normal HTML fragments for prose, code blocks, callouts, figures, formulas, and tables. The chapter shell, table of contents, section numbering, and navigation are added during the build.

Starter Workflow

Add sections to this file, add new chapter fragments when needed, and register those chapters in chapters-src/site-manifest.json.

Inline Emphasis

Use strong for content-important terms, emphasis for sentence-level stress, and text markers for short passages that should stand out in the current explanation. Marker colors are available as teal and pink, but meaning and priority should also be clear from the surrounding words.

inline emphasis
<strong>Classification</strong>
<em>focus</em>
<mark class="text-marker">standard marker</mark>
<mark class="text-marker text-marker-teal">teal marker</mark>
<mark class="text-marker text-marker-pink">pink marker</mark>
